# MetodyProgramowania---2022-2023
Repozytorium zawiera zestaw zadań programistycznych realizowanych w ramach kursu skupiającego się na algorytmice oraz efektywnym zarządzaniu danymi. Wszystkie rozwiązania zostały zaimplementowane w języku Java.

## Cel kursu
Głównym celem projektów było nabycie umiejętności projektowania, implementacji i analizy podstawowych struktur danych oraz algorytmów, a także rozwiązywanie problemów logicznych w sposób optymalny.

## Zakres materiału
Zgodnie z sylabusem kursu, zadania obejmują następujące zagadnienia:
- Struktury danych: Implementacja i wykorzystanie drzew, grafów oraz tablic haszujących.
- Algorytmika: Stosowanie wybranych technik konstrukcji algorytmów.
- Sortowanie i wyszukiwanie: Realizacja klasycznych algorytmów porządkowania i przeszukiwania zbiorów danych.
- Modelowanie problemów: Przekształcanie problemów opisanych w języku naturalnym na struktury danych i algorytmy.
- Dokumentacja: Pisemne opracowanie rozwiązań i analiza ich działania.

## Jak uruchomić projekt
1. Sklonuj repozytorium:
```
git clone https://github.com/TwójUsername/NazwaRepozytorium.git
```
2. Przejdź do wybranego folderu z zadaniem.
3. Skompiluj i uruchom program za pomocą IDE lub linii komend:
```
javac NazwaKlasy.java
java NazwaKlasy
```
